Projects

Memo App

This is a simple webapp created to store memoes. The app is composed by 3 pages:
  • The initial one allows to control the existing memoes, edit, and delete them;
  • The page on the left allows to create new memoes that will appear on the main page;
  • The page on the right is the settings page. At the moment, all it allows for is to activate the night mode.

The app is realised in plain Javascript (ES6), HTML, and CSS and it implements a webmanifest which makes it a fully working Progressive Wep App (PWA).

Grapher.js

This app was developed as part of my dissertation.

The idea behind this project was to create a tool that instructors could use to teach data structures and algorithms.

Grapher.js allows instructors to define:
  • the type of data structure to display as well as the functions that the student (or any other user) may call to interact with the data structure and modify it;
  • some pseudo-code and comments to display during the execution of each algorithm callable by the student;
  • some javascript code that can communicate with the software API (Grapher API) to animate the objects in the view (the code used by the algorithms callable by the student).
This was a very superficial explanation, but better instructions on how to use it can be found in the instructions page of the software.
