Memo App

This is a simple webapp created to store memoes. The app is composed by 3 pages:
  • The initial one allows to control the existing memoes, edit, and delete them;
  • The page on the left allows to create new memoes that will appear on the main page;
  • The page on the right is the settings page. At the moment, all it allows for is to activate the night mode.
The app is realised in plain Javascript (ES6), HTML, and CSS and it implements a webmanifest which makes it a fully working Progressive Wep App (PWA).
